# IB_cpp


## Download

```Bash
git https://github.com/tirelt/IB_cpp.git
cd IB_cpp
```

## Install

```Bash
mkdir tws_api
cd tws_api
wget https://interactivebrokers.github.io/downloads/twsapi_macunix.1019.04.zip
unzip twsapi_macunix.1019.04.zip
rm twsapi_macunix.1019.04.zip

```
